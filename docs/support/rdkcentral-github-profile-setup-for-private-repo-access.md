# RDK Central Github Profile Setup for Elevated Privileges or Private Repository Access

 [RDKCentral GitHub](https://github.com/rdkcentral/)
      

If you require elevated privilages or access to private repositories hosted on RDK Central Github https://github.com/rdkcentral/  then you are required to have an **rdkcentral user ID**, a **public GitHub account** and to **associate your public GitHub username with your RDK Central account**.

You must also ensure the email account associated with your rdkcentral account is a corporate email account (e.g. comcast, sky etc)

This is done by first creating an RDK Central account on https://wiki.rdkcentral.com and once created then enter your GitHub public username in your RDK Central wiki profile.

If you do not have a public GitHub account you can create one here: https://github.com

If you do not already have a rdkcentral id then please sign up here: https://wiki.rdkcentral.com/signup.action 

[RDKCentral Wiki](https://wiki.rdkcentral.com/)

<font color="red">PLEASE WAIT 15 MINUTES TO ALLOW YOUR ACCOUNT TO SYNCHRONISE WITH BACKEND SYSTEMS BEFORE EDITING YOUR PROFILE AS PER FOLLOWING STEPS</font>


**STEP 1: Sign In to rdkcentral wiki and navigate to your rdkcentral user profile**

https://wiki.rdkcentral.com/users/viewmyprofile.action


![RDKCentral Profile](https://github.com/rdkcentral/docs/blob/main/docs/support/images/RDKCentralProfile.png)


**STEP 2: Click on "Edit Profile"**

![RDKCentral User Profile](https://github.com/rdkcentral/docs/blob/main/docs/support/images/RDKCentralUserProfile.jpg)


**STEP 3: Enter your RDK Central public Github username:**

<font color="green">In addition if you are a Comcast or Sky employee and have a GHEC account then please also enter your GHEC account username.</font>

<font color="red">If you don't have a GHEC account leave it blank (whether you or a comcast/sky employee or not), if in doubt leave it blank.

Do not simply fill this field in with your comcast ID, it is only applicable if you have an active GHEC account, again f in doubt leave it blank and raise a CMF-SUPPORT ticket requesting the invite.</font>

![RDKCentral Edit Profile](https://github.com/rdkcentral/docs/blob/main/docs/support/images/RDKCentralEditProfile.png)

**STEP 4: Press Save**

Once saved the new settings will appear in your profile.

![RDKCentral Saved Profile](https://github.com/rdkcentral/docs/blob/main/docs/support/images/RDKCentralProfileAfterSave.png)


**STEP 5: GitHub invite**

If you are a member of Comcast GHEC or have raised a support ticket looking for access to a restricted repo then once you have done all the above you should receive a GitHub Invite to the **email account associated with your PERSONAL GithHub Account**, make sure you accept the invite. Valid GHEC users will automatically receive the invite and do not need to raise a support ticket unless they haven't received it within 24 hours. If you are already a member of the rdkcentral organisation by having accepted a previous invite then you will not receive a new invite, you will be granted access automatically assuming the other access criteria have been met.

If you don't have a GHEC account then you need to raise a support request specifying your public GitHub ID and the repo(s) you are looking for access to.

Comcast/Sky users do automatically have a GHEC account, in this case if you specify your Comcast user id as your GitHub ID and you do not have a valid/active GHEC account you will not get an automatic invite.

If you do not have a repository access request pending then you WILL NOT receive an invite, as the invites only apply for access for restricted repos and to those who have not accepted a previous invite.  

> a. if you haven't received the invite within 24 hours then please send a support request to support@rdkcentral.com
>>invites are not sent on Saturday/Sunday> 

> b.check https://github.com/settings/emails to see what your public GitHub primary email address is


If you have any issues please see [RDK Central Github Access Issues FAQ](https://wiki.rdkcentral.com/spaces/CMF/pages/355767084/RDK+Central+Github+Access+Issues+FAQ) before raising a support ticket also ensure you put your **public GitHub username** and any **repositories** you are having issues accessing in the ticket.
